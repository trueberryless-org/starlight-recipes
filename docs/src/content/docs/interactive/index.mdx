This page gives you a short overview of supported interactive features that automatically ship with Starlight Recipes and enhance the user experience and convenience.

## Setup

To use any of those features, you first need to add any [Astro Server Adapter](https://docs.astro.build/en/guides/on-demand-rendering/#server-adapters), which will allow rendering parts of your page on-demand.
Individual features may require additional steps to function properly. Check out the individual guides for details.
